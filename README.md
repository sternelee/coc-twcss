# coc-twcss

coc.nvim for tailwindcss, windicss, unocss

## Install

`:CocInstall coc-twcss`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-twcss-keymap)`

## Lists

`:CocList demo_list`
